# Portfolio Website - Shashi Kant

A modern, responsive portfolio website showcasing my projects, skills, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Smooth Scrolling**: Enhanced navigation experience
- **Interactive Elements**: Hover effects and animations throughout
- **Social Links**: Easy access to GitHub, LinkedIn, Instagram, and Email

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (Vanilla JS)
- Font Awesome Icons

## Sections

1. **Hero Section**: Introduction with call-to-action buttons
2. **About Me**: Personal background and interests
3. **Skills**: Technical skills and competencies
4. **Projects**: Showcase of completed projects with GitHub links
5. **Contact**: Contact information and social media links

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required!

## Customization

To customize this portfolio:

1. **Personal Information**: Edit the content in `index.html`
2. **Colors**: Modify CSS variables in `styles.css` (root section)
3. **Animations**: Adjust animation timings in `styles.css`
4. **Projects**: Add or remove project cards in the projects section

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
